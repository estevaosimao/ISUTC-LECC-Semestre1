# 1º Semestre - LECC (ISUTC)

Este repositório contém todo o material de suporte académico recolhido durante o 1º semestre do curso de LECC.

## Disciplinas incluídas:
- Exames de ALGA (Álgebra Linear e Geometria Analítica)
- Física 1
- Fundamentos de Programação
- Inglês Técnico
- Introdução à Engenharia Informática
- Matemática Geral

## Estrutura
Cada disciplina está organizada nas seguintes pastas:
- `Teorica/`: Slides e apontamentos.
- `Pratica/`: Guias de exercícios resolvidos.
- `Laboratorios/`: Códigos e relatórios.
- `Avaliacoes/`: Testes e mini testes.
- `Exames/`: Provas finais.
